4.1

Que es un "Stack" Tecnologico?

Es el conjunto de herramientas que usas dia a dia, integradas para maxima eficiencia. El Stack del Administrador IA incluye componentes especificos.

NUCLEO

Asistente IA Principal

ChatGPT Plus / Claude Pro / Gemini Advanced

GESTION

Tareas y Proyectos

Notion / Todoist / Asana / Monday

COMUNICACION

Correo y Mensajeria

Gmail + Gemini / Outlook + Copilot / Slack

DOCUMENTOS

Creacion y Edicion

Notion AI / Google Docs / Office 365

AUTOMATIZACION

Workflows

Zapier / Make / Power Automate

ANALISIS

Datos e Insights

ChatGPT Code Interpreter / Julius AI

Perfiles de Stack Recomendados

PERFIL 1

Administrador Google-centricos

  • Nucleo: Gemini Advanced
  • Workspace: Google Docs, Sheets, Gmail
  • Gestion: Notion para proyectos
  • Automatizacion: Zapier
  • Reuniones: Otter.ai

PERFIL 2

Administrador Microsoft-centricos

  • Nucleo: Microsoft Copilot
  • Workspace: Office 365 (Word, Excel, Outlook)
  • Gestion: Microsoft Planner/Project
  • Automatizacion: Power Automate
  • Reuniones: Teams con transcripcion nativa

PERFIL 3

Administrador Startup/Tech

  • Nucleo: ChatGPT Plus
  • Todo: Notion (todo en uno)
  • Comunicacion: Slack + integraciones
  • Automatizacion: Make (complejo)
  • Asincrono: Loom para videos

PERFIL 4

Presupuesto Limitado

  • Nucleo: ChatGPT gratuito + Claude gratuito
  • Workspace: Google Workspace gratuito
  • Gestion: Notion gratuito / Trello
  • Automatizacion: Zapier gratis (limitado)
  • Extra: Gemini en Gmail/Docs

FORMATO DE ENTREGA

Documento principal:

  • • PDF o Google Docs
  • • Extension: 5-8 paginas
  • • Incluir capturas de pantalla
  • • Anexos: prompts, flujos

Evidencias adicionales:

  • • Screenshots de herramientas configuradas
  • • Ejemplos de prompts ejecutados
  • • Diagramas de flujos implementados
  • • Comparativas antes/despues
